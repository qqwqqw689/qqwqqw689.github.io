Title

Xiaobei Laboratory has recently purchased several new servers for daily experimental data analysis. In order to let everyone use these machines reasonably and conveniently, you installed slurm and SCOW on them, and everyone used them happily.

Li Hua just joined Xiaobei Lab this year. His daily work involves analyzing large amounts of data. Li Hua wrote a program specifically for analyzing his data. He uses his laptop to run the program. However, Li Hua found that with the deepening of research, the data he needed to analyze became more and more complex; finally one day, Li Hua’s notebook “strike” - the “Fire Dragon No. The excessive heat generated during the continuous full-load work has caused itself to be on the verge of being burned, triggering the “strike” protection mechanism.

Therefore, Li Hua applied for a laboratory server account, and then he can run his program on the laboratory server, and his notebook can also get a good rest. This will save him a lot of money on repairs.

However, Li Hua found that every time he runs his own program on the cluster, he needs to enter a bunch of slurm-related commands. He thinks this is a bit cumbersome. He hopes you can help him write a script. Create a Job, allocate resources, and run Li Hua’s program. After the program is completed, the resource consumption of the program will be saved.

You see, because Li Hua promised this time that you no longer need you to write a thank-you letter for him that is addressed to you, you decided to do him this favor.

Submit request

Please submit a script file (allowed format: Shell script or Python script), which needs to meet the following requirements:

Have a shebang at the top of the file that indicates how to run the script (example: usually for a Bash script #!/bin/bash)

The script needs to complete the following tasks:

Run Li Hua’s program ( ) on the computing node ./program. This program receives a command line parameter. When calling the script you wrote, this parameter will also be passed in. If you write a shell script, you can use $ 1to node should be located at Partitioncompute , configured as one machine, one core ( -N1 -n1 -c1)

redirect the program’s output to output.dat

Use seff the command to view the resource usage of running this Job, and redirect the output to seff.dat

Note: Li Hua will not use it sbatch, so your program is executed directly on the login node!

Hint

In order to reduce your workload, Li Hua’s program will obtain the Slurm Job ID and output it to job_id.data file .

Li Hua’s program will not output to stderr, only to stdout.
In a shell script, pass the contents of a file as part of a command argument:

Use something command_name “$(cat filename.dat)”like

Use something cat filename.dat | xargs command_namelike

Solution

Explain

  1. #! -> shebang

    #!/bin/bash at the start or top of the script to instruct our system to use bash as a default shell.

    The shell is a program that takes commands from the keyboard and gives them to the operating system to perform.

  2. srun - Run parallel jobs

    -N

    Request that a minimum of minnodes nodes be allocated to this job.

    -n

    Specify only the number of processes to run with -n.

    -c

    Request that ncpus be allocated per process.

  3. $(command)

    Command substitution allows the output of a command to replace the command itself.
